# Agent Governance Workbench 中文说明

这是一个轻量级 AI Agent 治理工作台模板。

它不是新的 Agent 框架，而是帮助你管理一组 Agent：谁负责、风险多高、是否需要人工审批、结果证据在哪里、是否可以继续推进。

核心能力：

- 任务交接队列
- 风险分级
- 人工审批门禁
- 可审计结果卡
- 本地 Streamlit 看板
- 全部示例数据均为虚构数据

适合人群：

- 正在使用多个 AI Agent 的个人或团队
- 想要降低误操作、成本失控和隐私泄露风险的开发者
- 想给 Agent 工作流补上审批和审计层的人

这个公开仓库不包含任何真实个人数据、密钥、内部任务历史或私有路径。
