# Launch Copy

## GitHub Release

Title: `v0.1.0 - First public release`

Agent Governance Workbench is a local-first Streamlit dashboard and template kit for safer AI agent workflows.

Most agent frameworks help you build agents. This workbench helps you operate agent workflows with clearer ownership, scoped handoffs, risk levels, human approval gates, and evidence-backed result cards.

This first public release includes:

- A local Streamlit dashboard
- Synthetic example task queues, agent roles, workflows, and risk registers
- Handoff, result-card, approval-gate, and audit templates
- Release gates and privacy checklists
- A small local privacy scanner

The demo uses synthetic data only and does not require external APIs.

## Reddit / HN

I built a small local-first workbench for AI agent governance.

Most agent tools focus on building agents. I wanted a lightweight operating layer for the questions that come after that: who owns the next action, what needs approval, where is the evidence, and what should never be published?

The repo includes a Streamlit dashboard, synthetic sample queues, risk registers, approval-gate templates, result cards, release gates, and a simple privacy scanner. No external APIs are required for the demo.

Repo: https://github.com/yohojj/agent-governance-workbench
